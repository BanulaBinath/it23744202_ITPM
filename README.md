\## 📌 Project Overview

This project is an \*\*end-to-end test automation framework\*\* built using \*\*Playwright (Python)\*\*.



It automates testing of a \*\*Sinhala Transliteration Web Application\*\* by:

\- Reading test cases from an Excel file

\- Sending inputs to the web UI

\- Capturing actual outputs

\- Comparing with expected outputs

\- Writing results back to Excel (PASS / FAIL)



\---



\## 🎯 Key Features

\- 📊 Excel-driven automation

\- 🌐 Frontend UI testing

\- 🔄 Retry mechanism for dynamic responses

\- 🧠 Smart column detection

\- 🧾 Auto Excel updates:

&#x20; - Actual Output

&#x20; - Status (PASS / FAIL / UI Error)

\- ⚡ Headless \& headed execution

\- 🧩 Chat-based UI handling
